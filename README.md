## 說明:
```
這是某個 Online Judge 提供的題目
主要目的是建構資料結構的演算法題目
簡單說明:
  N = 題號, TOPIC = 題目標題, LANG = program language
  會依照題目放置在 N.TOPIC 的資料夾中
  N.TOPIC 會包含 main.LANG, 例如: main.c = C-language, C++ = cpp, Java = java, python = py ...(看這平台給什麼就打什麼)
  除此之外 N.TOPIC 會包含 test 資料夾, 而 test 中會依照 測資編號 (C) 放置
  輸入為 test.in.C
  輸出為 test.out.C
```
